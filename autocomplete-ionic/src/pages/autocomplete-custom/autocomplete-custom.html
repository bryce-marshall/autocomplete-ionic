Autocomplete Custom Colour This example demonstrates the use of custom sub-components for the popup Create and List sections. The bound model data-item is an object that exposes both name and rgb-colour properties. Both the allowCreate andautoAssign properties are set to "true", meaning that the user can create new colours and assign them either explictly (by clicking on a popup item) or implictly (by shifting focus to another control or document element after having entered a valid value).
